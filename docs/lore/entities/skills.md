# Skills

Skills are bad habits that slowly become almost useful. The Handler never assigns points; practice emerges from real outcomes, and level-up text is part of the joke.

| Skill | Canon identity | Practice source | Tone |
|---|---|---|---|
| Cardio | Running away while explaining why it was tactical. | Extraction attempts, returns, low-HP escapes. | Panic pretending to be fitness. |
| Hoarding | Advanced backpack theology. | Looting, extracting stuffed bags, valuable hauls, stash overflow. | Every object has resale destiny. |
| Hiding in Lockers | Furniture-adjacent stealth. | Surviving robot encounters, failed extractions, High danger survival, pocket saves. | Stillness as a lifestyle. |
| Signal Handling | Handler buttoncraft with almost believable confidence. | Successful Signal spends. | Procedure built from radio meddling. |

## Authoring Rules

- Skill text should sound like a career path no serious person would endorse.
- Mechanical benefits stay tiny and visible.
- Do not make skills feel like normal combat builds.
- Level-up lines should celebrate the wrong lesson learned from the right outcome.
